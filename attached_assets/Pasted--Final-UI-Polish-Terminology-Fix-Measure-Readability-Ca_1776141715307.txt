# 1. Terminology Fix (CRITICAL)

Replace ALL occurrences of:

* “Form I, II, III…”
  → with
* **“Measure I, II, III…”**

### Arabic pairing:

Where appropriate, use:

* **Measure I (الوزن الأول)**
* OR short version: **Measure I · وزن**

### Apply everywhere:

* card headers
* tooltips
* practice engine questions
* labels
* badges
* internal UI text

### Rule:

The system must consistently reflect **Measure (وزن)**, not Form.

# 2. Fix Crowded Card Header (Top-Right Corner)

## Problem

Too many elements stacked:

* theoretical / attested
* Measure label
* Roman numeral
* current badge
* English description

## Required fix

### Create clean structure:

#### Top metadata strip (small, quiet)

Contains:

* Measure label (Measure II, etc.)
* theoretical / attested (small, low contrast)
* current (very subtle)

#### Below it:

* English description (e.g. Causative, Reflexive)

#### Below that:

* Arabic verb (main focal point)

---

## Rules

* DO NOT stack multiple badges in the corner
* DO NOT let metadata compete with titles
* Move elements into horizontal strip if needed
* Keep corner visually calm

# 3. Enforce Language Hierarchy

## Arabic = Primary

* darker color
* stronger weight (500–600)
* largest visual element

## English = Secondary

* slightly lighter
* smaller than Arabic
* never equal weight

## Metadata = Tertiary

* lighter
* minimal visual presence

---

# 4. Fix Gray Text System (VERY IMPORTANT)

Current issue: too many gray shades + weak contrast

## Replace with ONLY 3 levels:

### Level 1 — Primary

* Arabic text
* strong, dark

### Level 2 — Secondary

* transliteration
* English support
* medium-dark gray (clearly readable)

### Level 3 — Tertiary

* metadata
* helper labels
* lighter gray but NOT faint

---

## Apply to:

### A. Transliteration

* darken further
* must be readable instantly

### B. Small labels

Fix:

* `.Pres`
* `Masdar`
* `Doer`

Actions:

* darken text
* remove overly faint styling
* keep consistent

### C. English example translations

* increase contrast
* ensure readability at normal zoom
