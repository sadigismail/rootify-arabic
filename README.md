# RootifyArabic

Arabic Roots & Verb Analysis Tool

## Overview
RootifyArabic is an advanced Arabic morphology engine designed to analyze roots, detect verb forms, and generate accurate conjugations.

The system supports:
- Root extraction
- Verb form classification (Form I–X)
- Weak and irregular verb handling
- Passive and active voice generation
- Morphological analysis for Arabic learners and linguists

## Features
- Smart root detection
- Full conjugation engine
- Arabic verb classification system
- Support for weak verbs (defective, hollow, assimilated)
- Passive voice generation
- Linguistic analysis pipeline

## Tech Stack
- Node.js
- TypeScript
- Custom Arabic Morphology Engine

## Project Structure
- /lib → core logic
- /data → lexicons
- /scripts → utilities
- /artifacts → builds
- /attached_assets → resources

## Purpose
This project aims to support:
- Arabic language learners
- Linguists
- AI-based language tools
- Educational platforms
